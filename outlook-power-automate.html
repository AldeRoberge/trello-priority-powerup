Power Automate → Outlook

Sans Entra, sans Trello Pro, sans serveur.

Qui fait quoi Power Automate crée l’événement Outlook et écrit [outlook-event-id]: … dans la description Trello.
Cerveau ne fait que masquer cette ligne dans le popup (bouton Afficher les métadonnées masquées). Sans étape « Update a card » dans le flux, vous ne verrez jamais de métadonnées.
Ouvrir Power Automate Guide GitHub

Flux (résumé)

  1. Déclencheur Trello : When a card is updated → échéance non vide.
  2. Si la description ne contient pas outlook-event-id → Create event, puis Update a card (description) en ajoutant : 
concat(
  triggerBody()?['desc'],
  '

[outlook-event-id]: ',
  body('Create_event')?['id']
)
  1. Sinon → lire l’id dans la description → Update event.
  2. Activer le flux → tester une carte avec échéance → rouvrir Cerveau → Afficher les métadonnées masquées.

« Exporter .ics » et « Connecter Outlook » (Graph) sont d’autres chemins. Pour Automate, seule l’écriture dans la description par le flux compte.
