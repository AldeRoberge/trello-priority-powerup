Automatiser Outlook (Power Automate)

Sync Trello → calendrier Outlook dans le cloud Microsoft. Aucun serveur, pas d’app Entra, pas besoin de Trello Pro (les champs personnalisés ne sont pas utilisés).

Ouvrir Power Automate Guide détaillé (GitHub)
Sens unique : Outlook ne met pas à jour Trello. Mapping des IDs dans Excel / OneDrive (gratuit) pour éviter les doublons.

1. Fichier Excel (mapping)

  1. Créez un classeur OneDrive / Excel Online, ex. Trello-Outlook-Map.
  2. Ligne d’en-têtes (formatez en Tableau) :
CardId EventId CardName
(lignes ajoutées par le flux)

2. Flux automatisé

  1. Power Automate → Créer → Flux de cloud automatisé.
  2. Déclencheur Trello : When a card is updated → votre tableau.
  3. Condition : Due date n’est pas vide.
  4. Excel : chercher une ligne dont CardId = id de la carte.
  5. Si pas de EventId → Create event (sujet, description, échéance), puis Add/Update row Excel avec l’id d’événement.
  6. Sinon → Update event avec cet EventId.

3. Variante sans Excel — métadonnées dans la description

Après création de l’événement, écrivez en bas de la description :

[outlook-event-id]: …

Ou le bloc commentaire <!--cerveau-meta … -->. Dans le popup Cerveau, ce texte est masqué par défaut (bouton Afficher les métadonnées masquées).

4. Tester

  1. Enregistrer et activer le flux.
  2. Mettre une échéance sur une carte test.
  3. Vérifier Outlook, puis renommer la carte (un seul événement).
